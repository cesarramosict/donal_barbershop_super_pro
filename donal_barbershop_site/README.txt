Donal BarberShop - Site vitrine professionnel (noir & blanc)
=================================================================

Structure :
- index.html : page d'accueil (hero fort, services, sélection de photos, contact).
- produits-prix.html : tarifs basés sur votre tableau de prix + texte sur les produits.
- photos-videos.html : galerie photos + vidéos (format 16/9, non déformé).
- equipe.html : présentation de l'équipe (texte personnalisable).
- contact.html : coordonnées + formulaire de contact.

Les images utilisées proviennent uniquement des photos fournies :
- salon-bw-main.jpg / salon-bw-alt.jpg : vues du salon en noir & blanc.
- salon-color-hero.jpg : vue couleur utilisée en fond de la hero (mais en niveau de gris).
- fade-client-1..6.jpg : coupes / fades.
- prix-board.jpg : photo du tableau de tarifs.

Les vidéos v1.mp4 et v2.mp4 sont dans assets/video et affichées dans la page photos-videos.

Pour tester :
1. Ouvrir ce dossier dans VS Code.
2. Utiliser l'extension Live Server sur index.html.
